Welcome to the model rail toolkit

Model Rail Toolkit

The Model rail toolkit is everything you need for making a digital model railway.

Requirements

To use this toolkit, you should have a computer capable of running Java 11+. The operating system can be Windows, Mac, or Linux.

You will also need to install JMRI on your computer.

Recommendations

It is recommended that you have at least a basic understanding of computers and electronics. Many projects also use electronic kits provided by MERG, which require a membership to buy kits. They also have an amazing community of experienced moderators that would be happy to help you if you require assistance. Please note that MERG kits require soldering, so a soldering iron is required.

Setup

To get started, clone the repository by opening a terminal and running:
